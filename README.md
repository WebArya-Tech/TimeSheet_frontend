# Timesheet Frontend

A modern Vite + React + TypeScript frontend for enterprise timesheet and attendance management.

## Features

- Role-based dashboard and navigation
- Daily timesheet entry and weekly submission flow
- Attendance calendar and leave management
- User administration and audit report views
- Responsive layout with reusable UI components

## Local development

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

## Build

```bash
npm run build
```

## Testing

```bash
npm run test
```

## Linting

```bash
npm run lint
```
